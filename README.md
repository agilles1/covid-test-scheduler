## Features 
- Create New Appointments
- Drag and Drop functionality for easy scheduling
- Auto generate the number of available appointments in a given hour
 
## Tech Stack
This web app makes use of the following:
- Ruby 
- Rails - MVC Web Framework
- PostgreSQL - Database
- Tailwind CSS

## Installing

- Clone this repo to your local machine `git clone <this-repo-url>`
- run `bundle install` to install required dependencies
- run `rails db:create` to create a database locally.
- run `rails db:migrate` to create tables into the database.
- run `rails db:seed` to create seed data.
- run `rails s` to run the server. 
- open the front end in your native browser vie the index.html file in the public folder
